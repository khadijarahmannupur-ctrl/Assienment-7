## 📌 Project Overview

**KeenKeeper** is a modern and user-friendly web application designed to help users maintain meaningful relationships with their friends. It allows you to track interactions like calls, texts, and video chats, ensuring you never lose touch with the people who matter most.

---

## 🚀 Key Features

### 👥 Friend Management

* View a list of all friends with detailed profiles
* Track last contact days and relationship status
* Clean card-based UI for better user experience

### ⚡ Quick Check-In

* Instantly log interactions (Call, Text, Video)
* Automatically adds entries to the Timeline
* Toast notifications for user feedback

### 📜 Timeline System

* Displays all interaction history
* Filter by interaction type (Call / Text / Video)
* Search by friend name or interaction type
* Sort by newest or oldest entries

### 📊 Friendship Analytics

* Dynamic Pie Chart using Recharts
* Visual representation of interaction types
* Real-time updates based on user activity

### 📱 Fully Responsive Design

* Works perfectly on mobile, tablet, and desktop
* Clean and modern UI following design guidelines
